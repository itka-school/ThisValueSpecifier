Тайны this в Java Script

this
ключевое слово, предоставляемое (зарезервированное) Java Script, которое указывает на контекст вызова
Контекст вызова
объект, "владеющий" вызванным методом. В Java Scrip им становится объект из которого вызван метод (объект перед точкой). Но, важно помнить, что есть особенности поведения this: в стрелочных функциях; когда функция вызывается оператором new; если функция находится в глобальном пространстве.
Значение this
Таким образом, в js значение this изменятся.
  • Если мы возовем this из глобальной области видимости, то его значением будет глобальный объект Window
  • Если мы используем this в классической функции, которая определена в глобальном объекте Window, то значением this станет глобальный объект Window или, если включить режим 'use strict', undefined.
  • Если мы используем this в стрелочной функции, которая определена в глобальном объекте Window, то она установит значение this из лексического окружения, т.е. из окружения, в котором эта функция была определена и в данном случае это также глобальный объект Window.
  • Если мы используем this в классической функции как метод объекта, то его значением будет объект, из которого этот метод вызван.
  • Если методом объекта мы назначим стрелочную функцию, то она возьмет в качестве this значение this из лексического окружения (т.е. из внешней области видиомсти этой стрелочной функции).
  • Если мы назначим this значением свойства объекта, объявленного литералом, то значением this будет глобальный объект Window
  • При вызове функции-конструктора оператором new значением this станет объект, который при данном вызове создает эта функция-конструктор.
  • В классах js значением this будет установливаться каждый новый создаваемый классом объект.
  • При подписке на событие элемента DOM this в обработчике события ссылается на элемент, на который был привязан обработчик. Если функция-обработчик не стрелочная функция. Стрелочная функция берет this из окружения.
  • Если метод или функция вызывается с помощью методов call(), bind() или apply(), то значением this в нем станет первый агрумент перечисленных методов.
